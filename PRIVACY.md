# PulseRelay プライバシーポリシー

最終更新日: 2026年6月18日

PulseRelay（以下「本アプリ」）の開発者である Shotaro Mizuochi（以下「開発者」）は、
本アプリにおける利用者情報の取扱いについて、以下のとおりプライバシーポリシーを
定めます。

## 1. 基本方針

本アプリは、Bluetooth Low Energy（BLE）対応の心拍計から心拍数等を取得し、
利用者が指定した OSC（Open Sound Control）送信先へ転送するデスクトップアプリです。

本アプリは、心拍データ、Bluetooth 機器情報、利用状況、診断情報を開発者の
サーバーへ自動送信しません。広告、行動分析、利用状況解析、クラッシュ解析その他の
トラッキング SDK も使用しません。

## 2. 本アプリがアクセスまたは処理する情報

本アプリは、その機能を提供するため、利用者の端末上で次の情報にアクセスし、
または処理します。

### 2.1 Bluetooth 機器および心拍データ

- 周辺の BLE 機器が発信する機器名、Bluetooth アドレス、サービス情報および
  接続状態
- Bluetooth SIG の標準 Heart Rate Service（UUID 0x180D）から取得する心拍数、
  センサー接触状態、消費エネルギーおよび RR 間隔（機器が提供する場合）
- BLE 接続、ペアリングおよび GATT 通信の結果

心拍数その他の健康状態に関係する情報は、その性質上、慎重な取扱いを要する情報です。
本アプリはこれらを、利用者が明示的に接続を開始した場合または自動接続を有効にした
場合に限り、端末上でリアルタイムに処理します。

本アプリは、心拍データの履歴や Bluetooth アドレスを設定ファイルへ保存しません。
処理中の値および一時的な診断情報はメモリ上に保持され、通常は本アプリの終了時に
消去されます。

### 2.2 OSC により送信される情報

OSC 出力が有効な場合、本アプリは心拍数を、利用者が設定したホスト、UDP ポートおよび
OSC アドレスへ送信します。既定の送信先は同一端末を示す `127.0.0.1:9000` です。

利用者が LAN 内またはインターネット上の別の送信先を指定した場合、心拍数はその
送信先の管理者またはサービス提供者に提供される可能性があります。この送信は
利用者の設定と操作に基づいて行われ、開発者は送信データを受領しません。

OSC は UDP を使用し、本アプリ自体は通信内容を暗号化しません。信頼できない
ネットワークや送信先に設定しないでください。送信先における情報の取扱いには、
当該送信先のプライバシーポリシーまたは利用条件が適用されます。

### 2.3 端末に保存される設定

本アプリは、次の設定を利用者の端末内の
`%APPDATA%\PulseRelay\settings.json` に保存します。

- 心拍データの取得方法
- 利用者が入力した BLE 機器名フィルター
- スキャン時間
- OSC の有効・無効、送信先ホスト、ポートおよび OSC アドレス
- 表示テーマおよび言語
- 起動時の自動接続、システムトレイおよび初回起動に関する設定

これらの設定は開発者へ送信されません。利用者は設定を変更するか、上記ファイルを
削除することで保存情報を変更または消去できます。アンインストール方法によっては
設定ファイルが端末に残る場合があるため、完全な削除を希望する場合は上記フォルダーを
手動で削除してください。

### 2.4 診断情報

本アプリは、動作状態、接続結果、直近の心拍数、OSC 送信結果、エラー内容および
BLE 機器情報を含む診断ログを、件数を制限したうえでメモリ内に一時保持します。
診断画面からログを消去でき、アプリ終了時にも消去されます。

「診断情報をコピー」する機能は、Bluetooth の MAC アドレス形式に一致する文字列を
マスキングします。ただし、コピーした情報には機器名、直近の心拍数、時刻、
エラー内容などが含まれる場合があります。開発者または第三者へ送信する前に、
利用者自身で内容を確認してください。診断情報は、利用者が自ら送信しない限り、
開発者へ送信されません。

## 3. 利用目的

本アプリが端末上で情報を処理する目的は、次のとおりです。

- BLE 心拍計の検出、接続、必要なペアリングおよび心拍測定値の取得
- 心拍数の画面表示および利用者指定の OSC 送信先への転送
- 接続状態、送信状態およびエラーの表示
- 利用者が選択した設定の保存
- 不具合の調査に利用できる診断情報の利用者への提供

開発者は、本アプリを通じて取得される心拍データを、広告、プロファイリング、
信用評価、雇用判断、保険判断または販売のために利用しません。

## 4. 開発者による収集および第三者提供

本アプリには、開発者が管理するアカウント機能、クラウド API、データベース、
広告サービスまたは解析サービスはありません。このため、本アプリの通常利用により
開発者が利用者の個人情報や心拍データを収集、保存または第三者提供することは
ありません。

次の場合は、本アプリ外で情報が取り扱われることがあります。

- 利用者が OSC の送信先として第三者の端末またはサービスを指定した場合
- 利用者が電子メール、GitHub Issues その他の手段で、問い合わせ内容や診断情報を
  開発者へ自発的に送信した場合
- Microsoft Store の取得、ライセンス、更新その他のプラットフォーム機能について、
  Microsoft が同社の方針に従って情報を取り扱う場合
- 法令に基づく開示が必要な場合

問い合わせ対応で開発者が受領した氏名、メールアドレス、問い合わせ内容および
添付された診断情報は、問い合わせへの回答、不具合調査、権利保護および法令遵守の
ためにのみ使用します。公開の GitHub Issues には、心拍データその他の機微な情報を
投稿しないでください。

## 5. 法的根拠

開発者が問い合わせ等を通じて個人情報を取り扱う場合、日本の「個人情報の保護に
関する法律」に従い、利用目的を特定し、その目的に必要な範囲で適正に取り扱います。

欧州経済領域（EEA）、英国または同様の制度が適用される地域の利用者について、
開発者が問い合わせ情報を取り扱う法的根拠は、状況に応じて、本人の同意、
問い合わせへの対応または契約前の措置、開発者の正当な利益（本アプリの安全性確保、
不具合対応および権利保護）、ならびに法的義務の履行です。

本アプリによる心拍データの処理は利用者の端末上で行われ、開発者は当該データを
受領しません。利用者は、BLE 接続または OSC 出力を開始・停止し、送信先を選択する
ことで、この処理を管理できます。

## 6. 保存期間

- 心拍データおよび診断ログ: 原則としてメモリ内だけで処理され、アプリ終了時まで
- アプリ設定: 利用者が変更または削除するまで端末上に保存
- 問い合わせ情報: 問い合わせへの対応、不具合調査、権利保護および法令上必要な
  期間に限って保存し、その後削除または匿名化

## 7. 安全管理措置

本アプリは、データ最小化、心拍履歴および Bluetooth アドレスの永続化禁止、
診断ログの件数制限、コピー時の MAC アドレス形式のマスキング、開発者サーバーへの
自動送信を行わない設計により、情報への不要なアクセスおよび漏えいのリスク低減に
努めます。

ただし、利用者の端末、Bluetooth 通信、ローカルネットワーク、OSC 送信先および
オペレーティングシステムの完全な安全性を保証するものではありません。利用者は、
端末のアクセス制御、OS の更新、信頼できる OSC 送信先の利用等の適切な安全対策を
講じてください。

## 8. 利用者の選択および権利

利用者は、次の方法で本アプリによる処理を管理できます。

- BLE 接続を開始または停止する
- 起動時の自動接続を無効にする
- OSC 出力を無効にする、または送信先を変更する
- 診断ログを消去する
- `settings.json` または `PulseRelay` 設定フォルダーを削除する
- 本アプリをアンインストールする

開発者が問い合わせ等を通じて利用者の個人情報を保有している場合、適用法令の
範囲内で、利用目的の通知、開示、訂正、追加、削除、利用停止、消去、第三者提供の
停止、同意の撤回その他の権利を行使できる場合があります。請求時には、本人確認に
必要な最小限の情報の提示をお願いすることがあります。

本アプリの通常利用では開発者が心拍データや設定を保有しないため、開発者側で
それらを開示または削除することはできません。端末上の情報は、上記の方法で利用者が
管理してください。

## 9. 子どものプライバシー

本アプリは子どもを主な対象として設計されたものではありません。判断能力が十分で
ない未成年者が本アプリを利用する場合は、保護者または法定代理人の確認および監督の
もとで利用してください。開発者は、本アプリを通じて子どもの情報を意図的に
収集しません。

## 10. 本ポリシーの変更

本アプリの機能、情報の取扱いまたは適用法令の変更に応じて、本ポリシーを改定する
ことがあります。重要な変更がある場合は、本ファイル、配布ページその他の適切な
方法で通知し、冒頭の最終更新日を更新します。
